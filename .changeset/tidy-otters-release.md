Publish `@github-tools/eve-extension` — a mountable [eve extension](https://eve.dev/docs/extensions) distribution for the GitHub tools, built on top of `@github-tools/sdk/eve`. Install with `pnpm add @github-tools/eve-extension` and mount it under `agent/extensions/`. See the package README and `examples/eve-extension-agent` for a runnable starter.
